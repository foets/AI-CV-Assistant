# CV Template Structure

This template defines the structure and formatting for generated CVs. The agent follows this structure exactly.

---

## CV STRUCTURE

### 1. Header

```
# [Full Name]  
**[Target Role Title]**  
[City, Country] | Email: [email] | Phone: [phone] | Telegram: [telegram]  
LinkedIn: [LinkedIn URL]

---
```

**Formatting**: Two spaces at end of lines 1-3 for line breaks. Blank line before `---`.

---

### 2. Professional Summary

```
## PROFESSIONAL SUMMARY

[3-4 sentences tailored to the target role]

---
```

**Content**: See agent prompt for tailoring rules. Maximum 4 sentences.

---

### 3. Professional Experience

```
## PROFESSIONAL EXPERIENCE

### [Job Title] | [Company Name]
*[Start Date] – [End Date] | [Location]*

- [Achievement with quantified impact]
- [Achievement with quantified impact]
- [Achievement with quantified impact]
- [Achievement with quantified impact]
```

**Rules**:
- One sentence per bullet, max 4 bullets per role
- Start each bullet with action verb (Led, Built, Delivered, Scaled, etc.)
- Blank line between date line and bullets
- Reverse chronological order

---

### 4. Education & Certifications

```
## EDUCATION & CERTIFICATIONS

**[Degree] — [Field of Study]** | [Institution], [Country]  
**[Certification]** | [Issuing Body]
```

**Rules**:
- Two spaces at end of each line for line breaks
- Omit years if not in user data (never write "N/A")
- Education first, then certifications

---

### 5. Skills & Languages

```
## SKILLS & LANGUAGES

**Competencies:** [skill1], [skill2], [skill3]  
**Soft Skills:** [skill1], [skill2], [skill3]  
**Tools:** [tool1], [tool2], [tool3]  
**Languages:** [language] ([proficiency]), [language] ([proficiency])
```

**Rules**:
- Two spaces at end of each line (except last) for line breaks
- One line per category, comma-separated values
- Use job terminology for skills user can demonstrate
- Include common tools from job if reasonable given user's background

---

## FORMATTING REFERENCE

| Element | Format |
|---------|--------|
| Main sections | `##` |
| Job titles | `###` |
| Dates/locations | `*italic*` |
| Labels | `**bold**` |
| Bullets | `-` |
| Section separators | `---` |
| Line breaks | Two trailing spaces |

**Length**: ~400-500 words (single page)
